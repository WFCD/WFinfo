Choose between the rewards showing up as an overlay on top of Warframe or as a separate window on a second screen

Choose your preferred hotkey to activate WFInfo's reward detection. Custom scaling factors are supported too! Set the scaling to what you use in game to get best results

Auto will use the logs created by warframe to detect when the reward screen is being displayed and run the program for you! Clipboard will copy the results into your clipboard allowing you to paste it into the chat for other players (Ctrl + V)
